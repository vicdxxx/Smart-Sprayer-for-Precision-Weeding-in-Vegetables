# Smart Sprayer

The code associated with the paper "Field Test and Evaluation of an Integrated Micro-Dosing Smart Sprayer for Precision Weeding in Vegetables".

# Abstract

Weed management in vegetable production systems is increasingly constrained by labor shortages, rising input costs, and the need to reduce herbicide use while avoiding crop injury. Precision, site-specific spraying offers a promising alternative to broadcast application; however, its effectiveness under real field conditions is often limited by unreliable weed detection, sprayer resolution, and timing inaccuracies. This study integrates an artificial intelligence-enabled smart spraying system that integrates real-time weed detection, plant tracking, and micro-dosing spray actuation for selective weed control in vegetable fields. The system employs a YOLOv10-small deep learning model trained on a diverse five-season crop-weed dataset comprising 14,186 images and 103,266 plant instances for robust weed detection, coupled with a ByteTrack-based tracking module for accurate spray timing. A micro-jet sprayer equipped with 12 independently controlled nozzles spaced at 1 cm intervals was mounted on an all-terrain vehicle (ATV) platform to target early-stage weeds, and an optimized multithreaded software architecture was implemented for system integration and real-time performance. Field experiments were conducted in a lettuce field to assess the weed detection and spraying performance. Video-based evaluation yielded a detection performance of 80.0% mAP@50, and field spraying tests achieved a weed hit rate of 84.6% at a platform forward speed of 0.914 km/h, representing a substantial improvement over previous system configurations. Results demonstrate that AI-driven detection and tracking, when integrated with a micro-dosing sprayer, can enable effective site-specific weed control under field conditions. This work advances the practical deployment of intelligent sprayer systems for precision weed management.
